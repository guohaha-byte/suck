# 傻瓜式操作流程（零基础也能跟着做）

这份文档假设你**什么都不会**。从头到尾照着每一步点就行。
做完后效果：屏幕显示光照亮度，手机/电脑能看到数据，还能远程开关板上两个灯。

> 全程大概 4 个阶段：① 装软件 → ② 插硬件 → ③ 改 4 行字 → ④ 点编译下载 → 测试。

---

## 阶段 0：你手上要有这些东西

- 普中麒麟 STM32F407 开发板（一块）
- 3.5 寸彩色屏（HX8357 那块）
- ESP8266 WiFi 小板（蓝色那个小模块）
- 下载器：ST-Link（一个 USB 小盒子，带 4 根线）
- 两根 USB 线、一台 Windows 电脑、一个 2.4G 的 WiFi（手机热点也行）

---

## 阶段 1：装软件（只装一次）

### 1-1 装 Keil（写程序、下载程序用的）
1. 找到并安装 **Keil MDK-ARM**（版本 5，安装包问老师/卖家要，或官网 keil.com 下载 `MDK5xx.exe`）。
2. 一路点「下一步」装完。
3. 装「器件包」：双击安装包附带的 **Keil.STM32F4xx_DFP.x.x.x.pack**，一路下一步。
   （没有这个包，编译会报找不到芯片。）

### 1-2 装 MQTTX（手机/电脑当遥控器和仪表盘用的）
1. 电脑浏览器打开 **https://mqttx.app** → 下载 Windows 版 → 安装。
2. 装完先放着，后面阶段 5 用。

> ✅ 本方案用的是**公共免费的 MQTT 服务器 `broker.emqx.io`**，你**不需要**自己搭服务器，省事。

---

## 阶段 2：插硬件（断电状态下插）

1. 把**彩色屏**插到开发板上写着 **LCD / TFTLCD** 的排针上（缺口/丝印对齐，别插反）。
2. 把 **ESP8266 模块**插到板上写着 **ESP8266 / WIFI** 的那一排针上。
3. **ST-Link** 接线到板子的 **SWD / JTAG** 口：
   - ST-Link 的 `SWCLK → 板子 SWCLK`
   - `SWDIO → SWDIO`
   - `GND → GND`
   - `3.3V → 3.3V`
4. ST-Link 插电脑 USB。开发板也用 USB 线供电，打开板子电源开关。
5. 光敏电阻（板上写 `LS1` 的小元件）是**板载的，不用接线**。

---

## 阶段 3：拿到程序代码

如果有人已经把代码给你了（一个文件夹），跳到阶段 4。

否则在电脑装个 **Git**（git-scm.com，一路下一步），然后桌面右键「Git Bash Here」，粘贴：

```bash
git clone <仓库地址>
cd suck
git checkout main
```

拿到后，文件夹里应该能看到 `Template.uvprojx` 这个文件——那就是 Keil 工程。

---

## 阶段 4：改 4 行字（最关键，必须做）

1. 进入代码文件夹 → `User` 文件夹 → 用**记事本**打开 **`main.c`**。
2. 找到最上面这几行（大概在第 16 行）：

```c
#define SSID          ""        // WiFi 名字
#define PASD          ""        // WiFi 密码
#define MQTTSV        ""        // 服务器地址
#define MQTTSV_PORT   ""        // 服务器端口
```

3. 把双引号 `""` 里**填上你的内容**，改成这样（示例）：

```c
#define SSID          "我的WiFi"          // ← 改成你家 WiFi 名字
#define PASD          "12345678"          // ← 改成你 WiFi 密码
#define MQTTSV        "broker.emqx.io"    // ← 照抄，公共服务器
#define MQTTSV_PORT   "1883"              // ← 照抄
```

> ⚠️ 注意：
> - 引号 `" "` 不能删，只改引号**里面**的字。
> - WiFi 必须是 **2.4G** 的（ESP8266 连不了 5G WiFi）。用手机开热点最稳。
> - 名字和密码要和手机连 WiFi 时输入的**一模一样**（区分大小写）。

4. 保存（Ctrl+S）。**改完这一步就够了，其它代码一律别动。**

---

## 阶段 5：编译 + 下载到板子

1. 双击 **`Template.uvprojx`**，用 Keil 打开。
2. 点工具栏的 **Rebuild 按钮**（一摞小方块的图标，或按快捷键 **F7**）。
   - 等下面黑窗口出现 **`0 Error(s)`** 就是成功。
   - 如果报错，多半是阶段 1-1 的「器件包」没装，回去补装。
3. 点 **Download 按钮**（向下箭头到芯片的图标，或按 **F8**）。
   - 出现 **`Application running ...` / `Programming Done`** 就是烧好了。
4. 按一下板子上的 **RESET（复位）按键**。

---

## 阶段 6：看屏幕（验证显示）

板子运行后，屏幕会显示：

```
        Light Monitor System
 Light :  ──          ← 光照亮度（数字会变）
 LED1  :  OFF
 LED2  :  OFF
 WiFi  :  Disconnected   ← 连上 WiFi 后变绿 Connected
 MQTT  :  Disconnected   ← 连上服务器后变绿 Connected
```

- 用**手电照**光敏元件 `LS1`：`Light` 数字**变大**；用**手遮住**：数字**变小**。
- 等十几秒，`WiFi` 和 `MQTT` 两行会先后变成绿色的 **Connected**。
- 如果一直是红色 Disconnected → 看「阶段 8 出问题怎么办」。

---

## 阶段 7：用 MQTTX 看数据 + 远程开关灯

### 7-1 连接服务器
1. 打开 **MQTTX** → 点左上角 **+ New Connection**。
2. 填：
   - **Name**：随便写，比如 `test`
   - **Host**：选 `mqtt://`，后面填 **`broker.emqx.io`**
   - **Port**：**1883**
   - 其它默认。
3. 点右上角 **Connect**，变绿就是连上了。

### 7-2 看板子上报的光照（每 3 秒一条）
1. 点 **+ New Subscription**（新建订阅）。
2. **Topic** 填 **`mcu`** → 点确认。
3. 等几秒，应该每 3 秒收到一条这样的消息：
   ```json
   {"light":78,"led1":0,"led2":1}
   ```
   `light` 就是当前亮度，遮挡光敏后再看数字会变。

### 7-3 远程开关板上的灯
1. 在 MQTTX 下方发送框：**Topic** 填 **`app`**。
2. 内容框里粘贴下面其中一条，点发送：

   | 想做什么 | 粘贴这个 |
   |---|---|
   | 开 LED1 | `{"led1":1}` |
   | 关 LED1 | `{"led1":0}` |
   | 开 LED2 | `{"led2":1}` |
   | 关 LED2 | `{"led2":0}` |
   | 同时控制 | `{"led1":1,"led2":0}` |

3. 发送后：板上对应的灯会**立刻亮/灭**，屏幕上 `LED1/LED2` 也会跟着变（绿=开，灰=关）。

> 🔔 公共服务器是大家共用的，万一你收到奇怪数据，是别人也在用 `mcu` 主题。
> 想要独占干净的主题（比如换成 `mcu_zhang`），需要改一点代码——直接找我帮你改。

## 阶段 8：出问题怎么办（对症查表）

| 现象 | 怎么办 |
|---|---|
| Keil 编译报错、找不到芯片 | 没装器件包，回阶段 1-1 装 `STM32F4xx_DFP` |
| 屏幕全白/不亮 | 屏没插紧，断电重插；还不行就是背光脚不同，找我改一行 |
| 屏幕花屏/颜色怪 | 接触不良，重插屏排针；多试一次复位 |
| WiFi 一直红色连不上 | ① 必须 2.4G WiFi；② 名字密码填错（大小写）；③ 用手机热点再试 |
| MQTT 一直红色 | WiFi 没连上时 MQTT 也连不上，先解决 WiFi |
| MQTTX 收不到 `mcu` 数据 | 确认订阅的是 `mcu`；确认屏上 MQTT 已 Connected |
| 发 `app` 灯不动 | 确认发送主题是 `app`，内容是 `{"led1":1}` 这种（含大括号和引号） |
| 光照数字方向反了 | 找我，把 `adc.c` 里取值改成不取反即可（一行） |

---

## 一句话总览

**装 Keil 和 MQTTX → 插屏/ESP8266/下载器 → 在 main.c 填 WiFi 和 `broker.emqx.io` → F7 编译 F8 下载 → 屏幕看亮度，MQTTX 订阅 `mcu` 看数据、发 `app` 开关灯。**

卡在哪一步，把屏幕显示或报错截图发我，我帮你看。
